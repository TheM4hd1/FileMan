**FileMan** is a `File Downloader` and `File Manager` for iOS Devices By **Swift 4**.

## Features:

* Supports multiple file downloading
* Supports Pause/Resume/Cancel downloads
* Supports Proxy (_under developing_)
* File Manager
* Music Player
* Video Player
* Image Viewer
* PDF/Document Reader
* Decompresser (zip)
